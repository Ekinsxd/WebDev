In order to create the grid, I split the rows into 4 sections and the columns into 4 sections. Using the following css: grid-template-columns: 3fr 1.5fr 4fr 1.5fr; grid-template-rows: 100px 1fr 1fr 100px; I was albe to allow the left side to have 30% of the area and the right side to have 70% of the area. On the right side i split up the areas into the remaining 7/10ths of the area into 1.5, 4, and 1.5. This allowed me to have area 5 and 6 be 15% of the area and area 4 and 7 allowed to occupy that 15% as well as the remaining 40% of the areaa. The tops and bottoms were trivial to implement.

Area 1 was simple, using a class on the html element, i allowed the element to occupy 100% of the area.

Area 2 was simple, only requiring text and a css element to limit the height of the image.

Area 3 is just text with headers and brs.

Area 4 is a table with table rows as chilren of the table, and table data as children of the table rows. The rows have alternating color by using tr:nth-child(odd) with the background set to some color.The hover effect was done in the same way, only selecting the tr subclass.

Area 5 is just an image with height and width set to 100%.

Area 6 is an unordered list with the list items being links. the links are created with the a tag and the href attribute.

Area 7 is just a large p tag with random b, i, and code tags sprinkled in there.

Area 8 is just a tags with href attributes linking to other websites. The images are contained as children of the a tags to inherit the href attribute.
